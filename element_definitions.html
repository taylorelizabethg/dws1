Element Definition Page

By: Elizabeth Taylor

h1
The main header of the page
h2
The second header of the page (smaller than h1)
h3
The third header of the page (smaller than h2)
h4
The fourth Header of the page (smaller than h3)
h5
The fifth Header of the page (smaller than h4)
h6
The sixth Header of the page (smaller than h5)
p
Indicates the beginning of a paragraph
b
Makes the text bold
i
Makes the text italic
sup
Makes the text a superscript
sub
Makes the text a subscript
br /
Makes the next word start on a new paragraph
hr /
Creates a break and inserts a horizontal line
strong
By default makes the text bold
em
By default makes the text italic
blockquote
Turns the paragraoh into a quote by separating it
q
Puts quotations around text
abbr
Will show the full name of an abbreviation when the curser crosses it in web page.
cite
Indicates the citing of a book or element by putting it in italics
dfn
Used to define terms. Safari and Chrome do not change text style but some browsers turn them into italics.
ol
Makes numbered lists
ul
Makes bulleted lists
li
Indicates items in a list
dl
Contains a list of definitions
dt
Indicated the term being defined
dd
Contains the definition
